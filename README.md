# Application d'Hébergement

## 1. Présentation du Projet et Objectif :
Application web fournissant des hébergements. Construction de la partie frontend avec React.

## 2. Durée du Projet :
Développé du 25 avril 2023 au 15 mai 2023, sur une période totale de 3 semaines.

## 3. Guide d'Installation et Utilisation :
1. Téléchargez les fichiers du projet.
2. Exécutez la commande `npm install`.
3. Entrez `npm start`.

## 4. Exemples :
Le projet est composé de 3 pages au total.

1. **Page Principale :**
   - Première page de l'application web, fournissant des informations générales aux utilisateurs.

2. **Page "Logement" pour chaque hébergement :**
   - Page fournissant des informations détaillées et des photos pour chaque hébergement.

3. **Page "A Props" :**
   - Page fournissant des informations supplémentaires ou des remerciements, offrant une variété de contenus aux utilisateurs.
